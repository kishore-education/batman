Dual Product Installation

Set up the Batman Extension or Batman-0.1 EXE based on your preferred workflow.

Track A: Batman Extension

Browser-first setup for extension users.

1

Download Extension Package

Get the official Batman Extension package from the release source.

2

Enable Developer Mode

Open Chrome at chrome://extensions/ and enable Developer mode.

3

Load Unpacked

Click Load unpacked and select the extension folder.

4

Open Portal and Use Shortcuts

Start your supported portal session and use extension hotkeys from the active tab.

Track B: Batman-0.1 (EXE)

Desktop executable setup for screenshot + auto-typing workflow.

1

Download batman-0.1.exe

Use the official executable link and save the file to a trusted local folder.

2

Run EXE and Login

Launch the EXE, complete sign-in if prompted, and verify credits are available.

3

Verify Hotkeys

Confirm screenshot, send-confirm, pause/resume, stop/reset, and mode toggle keys.

4

Run First Capture Flow

Take screenshots, confirm send, and validate that auto-typing output starts in your target input.
